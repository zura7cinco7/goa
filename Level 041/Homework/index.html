WHAT IS Front end

Frontend development focuses on building the user interface and user experience of a website or web application. It involves the use of HTML to structure the content, CSS to style the elements, and JavaScript to create interactive features.

HTML - HyperText Markup Language

HTML is the standard markup language used to create web pages. It provides the structure for web content, such as headings, paragraphs, links, images, and more. HTML defines the basic layout and content of a web page.

History of HTML

HTML was created by Tim Berners-Lee in 1991 as a way to organize and link documents on the internet. It has evolved over time to support multimedia content, dynamic elements, and responsive web design.

CSS - Cascading Style Sheets

CSS is used to style the presentation of a web page. It controls the layout, colors, fonts, and other visual elements of a page. CSS separates the content (HTML) from its design, making websites easier to maintain and modify.

History of CSS

CSS was introduced in 1996 to address the need for better web design control. It allowed web developers to separate the content from the design and create more flexible and dynamic layouts

JavaScript

JavaScript is a programming language used to create dynamic and interactive features on web pages. It allows for tasks like form validation, animations, and real-time content updates without needing to reload the page.

History of JavaScript

JavaScript was created by Brendan Eich in 1995 while working at Netscape Communications. It was initially designed to add simple interactivity to web pages but has since evolved into a full-fledged programming language used for both client-side and server-side development.
