ისტორიული ფაქტი

სამასი არაგველი

სამასი არაგველი — 1795 წლის 11 სექტემბერს კრწანისის ბრძოლაში გმირულად დაღუპულ მებრძოლთა რაზმი, რომელიც შედგებოდა სამასი ფშავ-ხევსურის და არაგვისპირელი გლეხისაგან. გადმოცემით, მათ ბრძოლის წინ შეჰფიცეს ერთმანეთს, რომ, ან გაიმარჯვებდნენ, ან ბრძოლის ველზე დაეცემოდნენ, რაც შეასრულეს კიდეც. მათი მამაცური თავდადება ფართოდ აისახა ხალხურ სიტყვიერებაში.

HTML

HTML (HyperText Markup Language) არის ვებგვერდების შექმნის ძირითადი ენა. HTML გვაძლევს საშუალებას ავაწყოთ ვებგვერდების სტრუქტურა და შინაარსი, მაგალითად ტექსტი, სურათები, ბმულები და სხვა ელემენტები.

CSS

CSS (Cascading Style Sheets) არის სტილური ენა, რომელიც გამოიყენება HTML დოკუმენტის ვიზუალური სახის დასაყენებლად. CSS-ის საშუალებით შეგვიძლია განვსაზღვროთ ფერები, ზომები, განლაგება, შრიფტები და სხვა ვიზუალური მახასიათებლები.

JS(JAVASCRIPT)

JavaScript არის პროგრამირების ენა, რომელიც გამოიყენება ვებგვერდებზე დინამიკური შინაარსის, ინტერაქტიულობის და ფუნქციონალის დასამატებლად. JavaScript მუშაობს ბრაუზერში და შეუძლია დაამუშაოს HTML და CSS, რეაგირება მოახდინოს მომხმარებლის მოქმედებებზე და შექმნას ინტერაქტიული ელემენტები.
